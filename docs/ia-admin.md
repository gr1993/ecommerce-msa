# 관리자 IA 문서

관리자용 화면/기능 구조를 서비스별로 정리한 문서입니다.

---

## 1 회원 관리
- 회원 조회

---

## 2 상품 관리 (Product-Service)

### 2-1. 상품 기본
- 상품 목록 조회
- 상품 검색 / 필터 (카테고리, 상태, 판매여부)
- 상품 등록
- 상품 수정
- 상품 상세 조회
- 상품 삭제 (논리 삭제)

### 2-2. 상품 구성
- 상품 옵션 그룹 관리
    - 옵션 그룹 등록 (예: 색상, 사이즈)
    - 옵션 값 관리 (Red, Blue / S, M, L)
- SKU 관리
    - SKU 목록
    - SKU별 가격
    - SKU별 재고 수량
    - SKU 활성/비활성
- 상품 이미지 관리
    - 대표 이미지
    - 상세 이미지
- 상품 상태 관리
    - 판매중 / 품절 / 판매중지

---

## 3 카탈로그 관리 (Catalog-Service)

- 카테고리 관리
    - 카테고리 트리 관리
- 전시 상품 관리
    - 전시 여부
    - 전시 순서
    - 메인 진열 영역 (메인 페이지 상단 배너, 추천 상품)
    - 이벤트/테마 상품 영역
    - 대표 상품 영역
- 검색 키워드 관리

---

## 4 주문 관리 (Order-Service)

- 주문 목록 조회
- 주문 상세 조회
- 주문 상태 변경
    - PENDING / PAID / CANCELLED / FAILED
- 주문 상품 목록 확인
- 주문자 정보 조회
- 주문 메모 / CS 메모

---

## 5 결제 관리 (Payment-Service)

- 결제 내역 조회
- 결제 상태 확인
- 결제 실패 내역
- 결제 취소(환불) 요청 목록
- PG 승인 정보 조회

---

## 6 배송 관리 (Delivery-Service)

- 배송 목록 조회
- 배송 상세
- 운송장 번호 등록
- 배송 상태 변경
- 배송사 연동 상태 확인

---

## 7 반품 / 환불 관리 (Return-Service)

- 반품 요청 목록
- 반품 상세
- 반품 승인 / 거절
- 환불 진행 상태 조회
- 재고 복구 여부 확인

### 운영자/쇼핑몰 운송장 발급
- 반품 승인 후 쇼핑몰에서 회수용 운송장 발급
    - 택배사 픽업 요청
    - 운송장 번호 안내
- 쇼핑몰/택배사가 회수용 운송장 자체 생성
- 장점/단점 정리

---

## 8 할인 / 프로모션 관리 (Promotion-Service)

- 쿠폰 목록
- 쿠폰 생성
- 쿠폰 발급 조건 설정
- 할인 정책 관리
    - 상품 할인
    - 카테고리 할인
- 이벤트 기간 설정

---

## 9 정산 관리 (Settlement-Service)

- 정산 대시보드
- 기간별 매출 조회
- 서비스별 매출
- 주문 기준 정산 내역
- 환불 반영 내역

---

## 10 운영 관리 (Operation-Service)

- 공지사항
