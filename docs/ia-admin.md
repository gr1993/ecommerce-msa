# 관리자 IA 문서

관리자용 화면/기능 구조를 실제 구현 기준으로 정리한 문서입니다.

---

## 1. 회원 관리 (User-Service)

- 회원 목록 조회 (페이지네이션)
- 이메일 / 이름 검색
- 회원 상태 필터링 (활성 / 비활성 / 정지)
- 회원 등급 필터링 (일반 / 실버 / 골드 / VIP)

---

## 2. 상품 관리 (Product-Service)

### 2-1. 상품 기본

- 상품 목록 조회 (페이지네이션)
- 상품명 검색
- 카테고리 필터링 (대 / 중 / 소 3단계)

**상품 등록**
- 기본 정보 입력 (상품명, 상품코드, 기본가격, 할인가격, 상품 설명, 판매 상태)
- 카테고리 지정 (최대 3개)
- 옵션 그룹 등록 (예: 색상, 사이즈)
- 옵션값 관리 (예: Red/Blue, S/M/L)
- SKU 자동 생성 (옵션 조합 기반)
- SKU별 가격 / 재고 / 활성 여부 설정
- 상품 이미지 업로드 (최대 10개)
- 대표 이미지 지정

**상품 수정**
- 상품 상세 조회
- 기본 정보 수정
- 옵션 그룹 / 옵션값 수정
- SKU 정보 수정
- 이미지 추가 / 기존 이미지 유지

### 2-2. 카테고리 관리 (Catalog-Service)

- 카테고리 트리 조회 (계층 구조, 최대 3단계)
- 카테고리 등록
- 카테고리 수정 (카테고리명, 전시 순서, 전시 여부)
- 카테고리 삭제 (하위 카테고리 없을 때만 가능)

### 2-3. 검색 키워드 관리

- 상품 목록 조회 (상품명 / 상태 필터)
- 상품별 검색 키워드 조회
- 검색 키워드 추가 (중복 체크 포함)
- 검색 키워드 삭제

---

## 3. 주문 관리 (Order-Service)

- 주문 목록 조회 (페이지네이션)
- 주문번호 검색
- 주문 상태 필터링 (CREATED / PAID / SHIPPING / DELIVERED / CANCELED / RETURN_REQUESTED 등)
- 주문 상세 조회 (모달) — 주문 정보, 주문 상품 목록, 배송 정보, 주문자 정보
- 주문 상태 변경
- 주문 메모(CS 메모) 수정
- 주문 취소 (CREATED / PAID 상태에서만, 취소 사유 입력)

---

## 4. 배송 관리 (Shipping-Service)

- 배송 목록 조회 (페이지네이션)
- 배송 상태 필터링
- 운송장 번호 검색
- 운송장 등록 (택배사 선택)
- 주문 상세 조회 (모달)

---

## 5. 반품 / 교환 관리 (Shipping-Service)

### 5-1. 반품 관리

- 반품 목록 조회 (페이지네이션)
- 반품 상태 필터링 (RETURN_REQUESTED / RETURN_APPROVED / RETURN_REJECTED / RETURNED)
- 주문번호 검색
- 반품 승인 (회수 수거지 정보 입력)
- 반품 완료 처리
- 주문 상세 조회 (모달)

### 5-2. 교환 관리

- 교환 목록 조회 (페이지네이션)
- 교환 상태 필터링 (EXCHANGE_REQUESTED / EXCHANGE_APPROVED / EXCHANGE_COLLECTING / EXCHANGE_RETURN_COMPLETED / EXCHANGE_SHIPPING / EXCHANGED / EXCHANGE_REJECTED)
- 주문번호 검색
- 교환 품목 상세 조회 (모달) — 원래 옵션 / 새 옵션 / 수량
- 교환 승인 (회수 수거지 정보 입력)
- 교환 배송 시작 (택배사 선택)
- 주문 상세 조회 (모달)

---

## 6. 운영 관리 (Promotion-Service)

### 6-1. 쿠폰 관리

- 쿠폰 목록 조회 (페이지네이션)
- 쿠폰 코드 / 이름 검색
- 쿠폰 상태 필터링
- 쿠폰 상세 조회 (발급 내역 포함)
- 쿠폰 등록 (할인 유형: 정률 / 정액, 최소 주문금액, 최대 할인금액, 유효기간)
- 쿠폰 수정

### 6-2. 할인 정책 관리

- 할인 정책 목록 조회 (페이지네이션)
- 정책명 검색
- 할인 정책 상태 필터링
- 할인 정책 상세 조회
- 할인 정책 등록 (적용 대상: 상품 / 카테고리 / 주문, 할인 유형: 정률 / 정액, 유효기간)
- 할인 정책 수정
